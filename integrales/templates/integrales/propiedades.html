Reglas y Propiedades de las Integrales

Aprende las reglas básicas que facilitan el cálculo de integrales.

Linealidad

Haz clic para ver ejemplo

∫[a·f(x) + b·g(x)] dx = a∫f(x) dx + b∫g(x) dx

Ejemplo: ∫(2x² + 3x) dx = 2∫x² dx + 3∫x dx

∫xⁿ dx = xⁿ⁺¹ / (n+1) + C

Ejemplo: ∫x³ dx = x⁴/4 + C
